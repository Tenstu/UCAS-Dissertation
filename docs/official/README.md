# 官方参考资料（UCAS 资环）

本目录是 `UCAS-Dissertation` 的官方参考资料单一权威位置（Single Source of Truth）。

## 文件清单

- `中国科学院大学资源与环境学位评定分委员会研究生学位论文撰写具体要.doc`  
  用途：资环分委员会“规范条款”原始依据（内容条款基线）。
- `中国科学院大学资环学科群研究生学位论文word模板.docx`  
  用途：Word 版式与结构实现基线（导出对齐与人工终稿整理基线）。
- `SHA256SUMS.txt`  
  用途：文件完整性校验，避免无感改动或误替换。

## 使用约定

- 不在项目根目录或其他目录保留副本，避免多处拷贝漂移。
- `scripts/export_docx.py` 默认优先从本目录发现参考 Word 模板。
- 若更新官方文件，请同步：
  1. 重算 `SHA256SUMS.txt`
  2. 更新根级 `README.md` 的引用说明
  3. 更新 `docs/ucas_resource_env_alignment_matrix.md` 对照说明
